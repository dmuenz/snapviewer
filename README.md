# 📷 SnapViewer
[SnapViewer](https://dmuenz.github.io/snapviewer/) is a simple web app for browsing and viewing [testthat](https://testthat.r-lib.org/) snapshots. You point it to a `_snaps` folder in an R package on your computer, and it shows a list of all snapshot files, e.g. `.md` files created via `expect_snapshot()` and `.svg` files created via `vdiffr::expect_doppelganger()`. Click any file to view it.

You can point SnapViewer to the `_snaps` folders for multiple packages, and you can easily toggle between them. SnapViewer will remember all previously accessed `_snaps` folders across browsing sessions. These features rely on the experimental JavaScript method [showDirectoryPicker](https://developer.mozilla.org/en-US/docs/Web/API/Window/showDirectoryPicker) which is implemented in Chrome and Edge but not in Firefox or Safari. So use Chrome or Edge for best compatibility. SnapViewer support for Firefox and Safari is in development but will not allow toggling between `_snaps` folders or remembering folders across sessions.
