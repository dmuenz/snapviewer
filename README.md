# 📷 SnapViewer
[SnapViewer](https://dmuenz.github.io/snapviewer/) is a simple web app for viewing [testthat](https://testthat.r-lib.org/) snapshots. You point it to a `_snaps` folder in an R package on your computer, and it shows a list of all snapshot files, e.g. `.md` files created via `expect_snapshot()` and `.svg` files created via `vdiffr::expect_doppelganger()`. Select any file to view it.

You can point SnapViewer to the `_snaps` folders for multiple packages, and easily toggle between them. SnapViewer remembers all previously accessed `_snaps` folders across browsing sessions. These features rely on the experimental JavaScript method [showDirectoryPicker](https://developer.mozilla.org/en-US/docs/Web/API/Window/showDirectoryPicker) which is implemented in Chrome and Edge but not in Firefox or Safari. So use Chrome or Edge for best compatibility.
